# Быстрый старт

## Локальный запуск (для тестирования)

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python app.py
```

3. Откройте в браузере: http://127.0.0.1:5000

## Доступы

### Админ-панель
- URL: http://127.0.0.1:5000/admin/login
- Логин: `admin`
- Пароль: `admin123`

### Пользователь
- Зарегистрируйтесь на главной странице

## Быстрое создание теста

1. Войдите в админ-панель
2. Создайте новый тест
3. Загрузите файл `example_test.json`
4. Тест готов!
